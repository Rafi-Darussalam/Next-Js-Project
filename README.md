# Lamborghini Fan Web

![Lamborghini Project Cover](/public/HomeHeroBg.png)

A modern, high-performance web application dedicated to showcasing the power and elegance of **Automobili Lamborghini**. Built with cutting-edge web technologies, this project features stunning UI/UX with responsive design, dynamic typography, and sleek components tailored for automobile enthusiasts.

## 🚀 Built With

- **Framework:** [Next.js](https://nextjs.org/)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/)

## ✨ Key Features

- **Immersive Homepage:** A striking hero section highlighting flagship capabilities (like 1015 CV max power, 0-100 km/h in 2.5s) with a sleek, dark-themed layout.
- **Fleet Showcase:** Dedicated sections to view the Lamborghini fleet with high-quality imagery and detailed performance stats.
- **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens for a seamless experience everywhere.
- **Custom Navigation:** A sleek, sticky navigation bar with a mobile-friendly menu and smooth routing.

## 📦 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites

Ensure you have Node.js installed on your machine.

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📂 Project Structure

- `src/app/`: Contains the Next.js App Router layouts, pages, and global CSS (`globals.css`).
- `src/Components/`: Modular and reusable React components categorized by feature:
  - `Home/`: Landing page sections (`HomeHero`, `HomeFirstSection`, etc.).
  - `Navbar/`: Navigation layout including mobile menus.
  - `Fleet/`: Components specific to the vehicle lineup presentation.
  - `Footer/` & `Customization/`: Additional modular interfaces.
- `public/`: Static assets and imagery used across the application.
